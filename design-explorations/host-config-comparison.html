Value pills, not scores

Each cell shows the actual hostConfig value. Booleans render as on/off; tri-states keep auto italic to mark "host decides"; absent fields are — with explanation.

Diverge gutter

A 3px coloured strip on the row's left edge marks rows where hosts disagree. Scanning the gutter is faster than reading every cell — it's the same trick git diff uses on the line gutter.

Override marker

When a host's value comes from hostCapabilitiesOverride rather than the hostStyle preset, we annotate beneath: "override · preset would be off". That preserves the [[advertise_equals_enforce]] distinction without an extra column.

Sketches use @mcpjam/design-system tokens. Section structure mirrors client-focus-tab-defs.tsx (Agent · MCP Protocol · Apps); adding a hostConfig field to the focus tab automatically gets it a row here.
