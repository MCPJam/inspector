The Registry is a curated catalog of pre-configured MCP servers. Browse available servers, connect with one click, and start testing immediately — no manual configuration needed.

## Key Features

- **One-Click Connect** — Add a pre-configured MCP server to your workspace instantly
- **OAuth Support** — Servers that require authentication guide you through an OAuth flow automatically
- **Verified Publishers** — Look for the verified badge to identify trusted server publishers
- **Star Favorites** — Star the servers you use most to show your support
## Getting Started

1. **Open the Registry tab** in the sidebar
2. **Browse the catalog** — Each card shows the server name, publisher, description, and supported client types
3. **Click Connect** to add the server to your workspace
4. **Authenticate if needed** — Servers with OAuth will open an authorization flow in your browser

Once connected, the server appears in your workspace and you can start using its tools in the [MCP Inspector](/inspector/tools-prompts-resources) or [App Builder](/inspector/app-builder).

## Server Cards

Each card in the registry displays:

- **Icon and name** — The server's identity at a glance
- **Publisher** — Who maintains the server, with a verified badge for trusted publishers
- **Auth badge** — Whether the server uses OAuth or requires no authentication
- **Star count** — How many users have starred the server
- **Description** — A short summary of what the server does

## Connecting a Server

Click **Connect** on any server card to add it to your current workspace. If the server uses OAuth, a browser window opens for you to authorize access. Your credentials are never shared with other workspace members — each person authenticates independently.

Some registry servers are listed with both an App and a Text variant. App connects the server with widget/UI output enabled, while Text connects it with plain text responses only. When both are available, a dropdown lets you pick which variant to connect.

<Note>
After connecting an OAuth server from the registry, MCPJam automatically redirects you to the App Builder so you can start testing right away.
</Note>

## Quick Connect

Featured registry servers also appear as a Quick Connect row at the top of the Servers tab. Click any pill to connect instantly without switching to the full Registry view.

## Starring Servers

Click the star icon on any card to mark it as a favorite. The star count is visible to everyone, helping surface popular servers in the catalog.
