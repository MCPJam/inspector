The [**@mcpjam/sdk**](https://www.npmjs.com/package/@mcpjam/sdk) helps you test MCP servers from code.

You can use it to:

- call tools directly
- connect servers to an LLM
- write tests
- run evals many times and measure results

## Who is this for?

- **MCP server developers**: Check that tools, prompts, and resources work
- **MCP client developers**: Build apps that talk to one or more servers
- **Teams running evals**: Measure quality and catch regressions

## Main Jobs

## Installation

```bash
npm install @mcpjam/sdk
```

## Quick Example

```typescript
import { MCPClientManager, TestAgent, EvalTest } from "@mcpjam/sdk";

// 1. Point to a server
const manager = new MCPClientManager({
  myServer: {
    command: "npx",
    args: ["-y", "@modelcontextprotocol/server-everything"],
  },
});

await manager.connectToServer("myServer");

// 2. Create an agent that can use the server's tools
const agent = new TestAgent({
  tools: await manager.getTools(),
  model: "anthropic/claude-sonnet-4-20250514",
  apiKey: process.env.ANTHROPIC_API_KEY,
});

// 3. Ask something and inspect the result
const result = await agent.prompt("Add 2 and 3");
console.log(result.toolsCalled());     // ["add"]
console.log(result.e2eLatencyMs());    // 1234

// 4. Run the same kind of check many times
const test = new EvalTest({
  name: "addition",
  test: async (agent) => {
    const r = await agent.prompt("Add 2+3");
    return r.hasToolCall("add");
  },
});
await test.run(agent, { iterations: 30 });
console.log(`Accuracy: ${(test.accuracy() * 100).toFixed(1)}%`);

await manager.disconnectServer("myServer");
```
