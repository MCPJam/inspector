MCPJam Inspector is a local workbench for MCP servers and apps.

Use it to:

- connect a server
- see its tools, resources, and prompts
- run tools by hand
- test app widgets locally
- debug auth and protocol issues

It helps you test everything in one place instead of juggling tunnels, custom scripts, and client apps.

## What You Can Do

| Capability           | Description                                                                                                                                                                                                                                                                                        |
| -------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| ChatGPT Apps SDK     | Build and test ChatGPT apps locally, including widget behavior and tool calls. [Read more](https://www.mcpjam.com/blog/app-builder) |
| MCP ext-apps (Claude)   | Build and test MCP Apps locally, including the main JSON-RPC UI messages. [Read more](https://www.mcpjam.com/blog/mcp-apps-preview)                                                                   |
| OAuth Debugger       | Step through your OAuth flow and inspect each request and response. [Read more](https://www.mcpjam.com/blog/oauth-debugger)                          |
| LLM playground       | Chat with your MCP server using built-in models or your own API key. [Read more](https://www.mcpjam.com/blog/frontier-models)                               |
| MCP server debugging | Connect to local or remote servers, run tools manually, and inspect logs.                                                                             |
| Server info          | View server icons, version, capabilities, instructions, and app metadata. [Read more](https://www.mcpjam.com/blog/server-instructions)                                                                                                                            |
