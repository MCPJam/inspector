## Quick Start

Run MCPJam Inspector:

```bash
npx @mcpjam/inspector@latest
```

When it starts, you'll get a `localhost` link. Open it in your browser.

<Note>
  Don't have Node.js installed? Download our desktop app for [Mac](https://github.com/MCPJam/inspector/releases/latest/download/MCPJam.Inspector.dmg) or [Windows](https://github.com/MCPJam/inspector/releases/latest/download/MCPJam-Inspector-Setup.exe), or use the [hosted app](https://app.mcpjam.com) in your browser — no installation needed.
</Note>

## Connect Your First MCP Server

Click "Add server". For a quick test, use the sample MCP server below:

```bash
npx -y @modelcontextprotocol/server-everything
```

<Frame>
  <img src="/images/everything-demo.png" alt="Server everything demo" />
</Frame>

Click connect. If it worked, you should see "Connected".

You can keep adding more servers later. MCPJam supports multiple connections at once.

## Test an MCP Tool

Open the "Tools" tab. This is where you can run a tool by hand and see what it returns.

Try the `add` tool:
1. Select the `add` tool from the list
2. Set parameters for `a` and `b`
3. Click "Execute"
4. Check the result at the bottom
