## Connect your first MCP server

Go to the **Servers** tab and add this server — it's Excalidraw, a diagramming tool with a built-in MCP app:

```bash
npx -y @excalidraw/mcp-server
```

Once connected you'll see a green status indicator. You're live.

## Open App Builder

Head to the **App Builder** tab and try this prompt:

```
Draw an MCP architecture diagram
```

Watch the diagram render live inside MCPJam — no ngrok, no subscription needed.

## This is just the beginning
